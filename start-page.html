Latest News

Praxis LIVE v3.4.0 - released December 21 2017

This release brings support for adding third-party libraries to projects as well as a range of bug fixes.

Read release notes

IMPORTANT - GStreamer is no longer included for OSX and Windows and must be installed separately if you want to play videos or use video capture. Windows users make sure to select the full rather than typical installation if you want to use capture devices.

  • Windows 32bit - https://gstreamer.freedesktop.org/data/pkg/windows/1.12.4/gstreamer-1.0-x86-1.12.4.msi
  • Windows 64bit - https://gstreamer.freedesktop.org/data/pkg/windows/1.12.4/gstreamer-1.0-x86_64-1.12.4.msi
  • Mac OSX - https://gstreamer.freedesktop.org/data/pkg/osx/1.12.4/gstreamer-1.0-1.12.4-x86_64.pkg

Praxis LIVE v3.3.0 - released October 13 2017

This release concentrates on useful new functionality for (live) coding, along with support for OBJ files, keyboard/mouse input, data processing, and bug fixes.

Praxis LIVE v3.2.0 - released August 18 2017

This release includes updates for Processing, NetBeans and TinkerForge; enhanced keyboard controls and actions in the graph editor; various minor code improvements; and, a range of bug fixes.

Praxis LIVE v3.1.0 - released April 30 2017

This release adds support for sharing OpenGL images via Syphon/Spout, OpenGLES rendering for the Raspberry Pi, PFont & text rendering, and PShape & SVG support.

Praxis LIVE v3.0.0 - released October 27 2016

A first release of version 3 is now available with lots of exciting new features. Full release documentation and new examples are currently being worked on - for changes and roadmap see this page.
