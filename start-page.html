Praxis LIVE v2.3.2 - updated May 13 2016

This release adds a new video:gl:p2d component for improved 2D OpenGL rendering, adds comments and custom colouring into the graph editor, and includes a range of important bug fixes.

Praxis LIVE v2.2 - released October 28 2015

This release updates the OpenGL pipeline to Processing v3. While giving improved performance this may cause regressions - please update carefully and report back any issues.

This release also adds optional GStreamer 1.x support, simple tracker-style table editing, and some additional components.

Praxis LIVE v2.0 - released March 13 2015

Praxis LIVE v2 is a major revision bringing a wealth of new features.

  • New OpenGL video pipeline - based on Processing.
  • Distributed hubs - run a project across multiple computers, or multiple processes on one machine eg. for better framerates or lower-latency audio.
  • New live-code support, using annotations to define ports and controls.
  • Fork All - majority of built-in components re-built using live-code support, if a built-in component doesn't quite do what you want, rewrite it on the fly!
  • Almost full access to the underlying Processing API, including 3D rendering.
  • New code editor - based on the NetBeans editor with full code completion and error highlighting.

Keep an eye on the website, issue tracker or Twitter feed for known issues and updates. Please report any problems.

Thank you for contributing to the success of Praxis LIVE.
